# git hub에 자료 올리는 법

1)준비

cd front\blog 에 들어간 뒤 yarn develop 을 실행하여 localhost:8000 을 접속할 수 있게 만든다.
(로컬확인용)


2)만들어진 페이지 업로드 하기

**git add .**

**Git commit -m “ 쓸 내용”**

**git push origin develop**

**yarn develop**

을 입력하여 업로드를 확인한다.
